WHAT THIS TOOL DOES
Defines damping behaviour using 6 velocity/force points and fits:
Linear, Quadratic or Cubic curves.

DATA REQUIRED
- Geometry values
- V1–V6 velocities
- Adj-only and Full forces for each velocity

MODEL CHOICE
Linear: simple
Quadratic: typical suspension
Cubic: complex, allows knees and inflection
